Gamebira adalah sebuah game edukasi Bahasa Inggris untuk anak-anak Sekolah Dasar.
<br>
Gamebira is an English educational game designed for elementary school children.
<br>
Gamebira（ゲームビラ）は小学生向けの英語知育ゲームです。
